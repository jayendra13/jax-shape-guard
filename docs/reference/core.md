# Core

Core abstractions for symbolic dimensions and unification.

## Dim

Symbolic dimension that unifies at runtime.

::: shapeguard.core.Dim

## Batch

Specialized dimension for ML batch sizes.

::: shapeguard.core.Batch

## UnificationContext

Tracks dimension bindings during shape checking. See [Unification](../concepts/unification.md) for a conceptual explanation.

::: shapeguard.core.UnificationContext

## Binding

Record of a dimension binding with source info.

::: shapeguard.core.Binding
